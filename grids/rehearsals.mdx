---
title: 'Rehearsals'
description: 'Admin grid for viewing and managing rehearsal details.'
---

This grid provides administrators with a table view of rehearsal data, including date, conductor, soloists, and venue. 

Features include:

- Columns for date, conductor, soloists, and venue
- Sorting and filtering capabilities
- Admin actions for adding, editing, and deleting calls (permissions-based)
