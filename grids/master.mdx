---
title: 'Master'
description: 'Admin grid for viewing and managing both concert and rehearsal details.'
---

This grid provides administrators with a table view of both concert and rehearsal data in a single list, including date, conductor, soloists, and venue. 

Features include:

- Columns for date, conductor, soloists, and venue
- Sorting and filtering capabilities
- Admin actions for adding, editing, and deleting calls (permissions-based)
