---
title: 'Archive'
description: 'Admin grid for viewing and managing archive concert details.'
---

This grid provides administrators with a table view of archive (past) concert data, including date, conductor, soloists, and venue. 

Features include:

- Columns for date, conductor, soloists, venue, and type
- Sorting and filtering capabilities
- Admin actions for adding, editing, and deleting concerts (permissions-based)
