# Workshop Leaders

The Workshop Leaders form allows admin users to manage the list of workshop leaders associated with the orchestra. Use this form to add, edit, or remove workshop leader records.

## Purpose
This form is designed for admin users to:
- Add new workshop leaders
- Edit existing workshop leader details
- Remove workshop leaders when necessary

## How to Use the Workshop Leaders Form

1. **View Workshop Leaders**: The form displays a list of current workshop leaders, including their names, instruments, and contact details.
2. **Add Workshop Leader**:
   - Click the **Add Workshop Leader** button.
   - Fill in the required fields (see below).
   - Click **Save** to add the new workshop leader.
3. **Edit Workshop Leader**:
   - Select a workshop leader from the list.
   - Update their details as needed.
   - Click **Save** to apply changes.
4. **Remove Workshop Leader**:
   - Select the workshop leader to remove.
   - Click the **Delete** or **Remove** button.
   - Confirm the action when prompted.

## Key Fields
- **Firstname**: Workshop leader's first name.
- **Surname**: Workshop leader's surname.
- **Display Name**: Name to display publicly.
- **Address 1 / Address 2 / Town / City / County / Postcode / Country**: Address details.
- **Home Phone**: Home phone number.
- **Mobile Phone**: Mobile phone number.
- **Email**: Email address (required).
- **Instrument 1 / Instrument 2**: Main and secondary instruments (dropdowns).
- **Note**: Additional notes.
- **Nationality**: Nationality (dropdown).
- **Diary Service**: Diary service used (dropdown).
- **Gender**: Gender (radio buttons).
- **Has CRB?**: Checkbox for DBS/CRB check.
- **CRB Expiry Date**: Expiry date for DBS/CRB check.
