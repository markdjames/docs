# artist-agents

The Artist Agents form allows admin users to manage the list of artist agents associated with the orchestra. Use this form to add, edit, or remove artist agent records.

## Purpose
This form is designed for admin users to:
- Add new artist agents
- Edit existing artist agent details
- Remove artist agents when necessary

## How to Use the Artist Agents Form

1. **View Artist Agents**: The form displays a list of current artist agents, including their names, agencies, and contact details.
2. **Add Artist Agent**:
   - Click the **Add Artist Agent** button.
   - Fill in the required fields (see below).
   - Click **Save** to add the new artist agent.
3. **Edit Artist Agent**:
   - Select an artist agent from the list.
   - Update their details as needed.
   - Click **Save** to apply changes.
4. **Remove Artist Agent**:
   - Select the artist agent to remove.
   - Click the **Delete** or **Remove** button.
   - Confirm the action when prompted.

## Key Fields
| Field          | Description                                                                 |
|----------------|-----------------------------------------------------------------------------|
| **First Name** | First name of the artist agent.                                            |
| **Surname**    | Last name of the artist agent.                                             |
| **Agency**     | Name of the agency the agent represents.                                   |
| **Phone**      | Contact phone number for the agent.                                       |
| **Direct Line?**| Indicate whether the phone number is a direct line.                       |
| **Email**      | Contact email address for the agent.                                      |
| **Notes**      | Any additional notes or relevant information about the agent.             |
