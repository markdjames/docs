---
title: Patches by Player
description: View orchestral patches by individual player
---

This section provides an interface for administrators to view patches by individual player. 

Features include:

- Select a player to view all patches they are assigned to
- Filter patches by scope: future, past, or all
- View details for each patch, including title, date, calls, and player participation statistics
- Visual indicators for player assignment completeness (e.g., color-coded counts)
- Data table with sorting and admin actions (edit, add)
