---
title: Patches Overview
description: View all orchestral patches
---

The Patches Overview provides administrators with a comprehensive view of all orchestral patches (concert and education assignments). Key features include:

- Tabbed interface to switch between Concerts and Education patches
- Filter patches by scope (future, past, all)
- View patch details such as title, date, calls, player participation, and publication status
- Visual indicators for patch completeness and publication (e.g., color-coded status)
- Data tables with sorting and admin actions (edit, add)
- Integration with backend data for real-time updates
