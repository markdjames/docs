# Personal Details Form

Use this form to update your personal profile information. Please follow these instructions:

1. **First name**: Enter your given name. This field is required.
2. **Last name**: Enter your surname/family name. This field is required.
3. **Performance name (if different)**: If you use a different name for performances, enter it here. This field is required.

Click **Save** to submit your changes. If the update is successful, you will see a confirmation message. If there is an error, an error message will be displayed.
