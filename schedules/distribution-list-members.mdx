This page provides a distribution list of all members of the orchestra. It includes contact details such as email addresses and postal addresses.

## Managing the Distribution List
Members are automatically added to the distribution list based on their profile settings. If a player is marked as a member of the orchestra they will be included in the distribution list.

## Viewing the Distribution List
The distribution list can be viewed in a table format, divided by orchestra section, which includes the following columns:
- **Name**: The full name of the player.
- **Email**: The email address of the player.

There are also two links at the top of the page:
- **Open Distribution Address List**: This link opens a new page with the address list of all members who have been marked as members of the distribution list, formatted for printing.
- **Open Full Address List**: This link opens a new page with the full address list of all members, formatted for printing.
- **View Distribution List Emails**: This link opens a text box containing the email addresses of all members who have been marked as members of the distribution list, which can be copied and pasted into an email client.
- **View Full List Emails**: This link opens a text box containing the email addresses of all members, which can be copied and pasted into an email client.
